# 🎮 Читаем вместе

Веб-игра для обучения детей 4–7 лет чтению на русском языке.

Ребёнок видит слово → читает его → выбирает правильную картинку среди 4 вариантов. Ловушки основаны на типичных ошибках чтения.

---

## 🚀 Как запустить

### 1. Запусти сервер картинок

Сервер с картинками должен работать на порту **8080**:

```bash
# В папке open-word-images
npm run dev
# или
npx http-server -p 8080 -c-1 --cors
```

### 2. Запусти игру

```bash
# В папке ninachit
python -m http.server 3000
# или
npx serve -l 3000
```

Открой в браузере: **http://localhost:3000**

## 📖 Как заполнить curriculum.json

Смотри **curriculum-schema.json** — там описания всех полей.

### Схема урока

```json
{
  "stages": [
    {
      "name": "Название этапа",
      "description": "Описание целей этапа",
      "lessons": [
        {
          "runtime": {
            "target_word": "Слово для чтения (русский)",
            "image_words": ["картинка1", "картинка2", "картинка3", "картинка4"],
            "correct_image_index": 0
          },
          "meta": {
            "pedagogical_goal": "Чему учит этот урок",
            "trap_logic": "Почему ребёнок может ошибиться",
            "parent_note": "Что сказать при ошибке",
            "theme": "Тема урока",
            "difficulty": 3
          }
        }
      ]
    }
  ]
}
```

### Описание полей

#### `runtime` — данные для игры

| Поле | Тип | Описание |
|------|-----|----------|
| `target_word` | строка | Слово, которое ребёнок читает (русский). Например: `КОТ`, `МОЛОКО`, `ВЕЛОСИПЕД` |
| `image_words` | массив 4 строк | Слова для картинок (англ.). Имена файлов без `_latest.png`. Первая — правильная, остальные — ловушки. Например: `["cat", "whale", "house", "juice"]` |
| `correct_image_index` | 0–3 | Индекс правильной картинки в `image_words` |

#### `meta` — педагогический контекст

| Поле | Тип | Описание |
|------|-----|----------|
| `pedagogical_goal` | строка | Чему учит этот урок. Например: «Чтение слов из 3 букв», «Закрытый слог», «Стечение согласных» |
| `trap_logic` | строка | Описание ловушек — почему ребёнок может выбрать неправильную картинку. Например: «whale (кит) — типичная ошибка О/И», «elephant (слон) — пропустил букву Т» |
| `parent_note` | строка | Подсказка для родителей — что сказать ребёнку после ошибки. Показывается в жёлтой зоне после ответа. Например: «Посмотри, тут буква О, а не И. Кит это КИТ» |
| `theme` | строка | Тема урока для контекста: Еда, Животные, Семья, Природа, Мебель, Школа... |
| `difficulty` | 1–9 | Уровень сложности. 1–2 = гласные и слоги, 3–4 = короткие слова, 5–6 = закрытые слоги, 7–9 = длинные слова с приставками |

> **ID и счётчики** восстанавливаются автоматически по индексу — их заполнять не нужно.

## 🎮 Логика игры

1. Загрузка курса → `fetch('curriculum.json')`
2. Загрузка прогресса → `localStorage`
3. Показ слова + 4 картинки
4. Обработка ответа → ✅/❌ + подсказка
5. Сохранение прогресса
6. Переход к следующему уроку

### Жизни

- 3 сердца ❤️❤️❤️
- При ошибке — теряется 1
- При 0 — экран завершения

---

## 💾 Прогресс

Хранится в `localStorage` под ключом `reading_game_progress`:

```json
{
  "stageIdx": 0,
  "lessonIdx": 0,
  "lives": 3
}
```

Сброс: кнопка ⚙️ в правом верхнем углу.

## 🖼️ Картинки

Загружаются с локального сервера:
```
http://127.0.0.1:8080/styles/flat/{word}_latest.png
```

Репозиторий: **krivich/open-word-images**

## 📊 Этапы курса

| Этап | Уроки | Описание |
|------|-------|----------|
| **A1** | 1–10 | Гласные, открытые слоги |
| **A2** | 11–25 | Первые слова (3 буквы) |
| **B1** | 26–40 | Закрытые слоги, стечения |
| **B2** | 41–50 | Длинные слова, приставки |

---

## 🔧 Тестирование

Сбросить прогресс в консоли:
```javascript
localStorage.removeItem('reading_game_progress');
location.reload();
```

Установить урок напрямую:
```javascript
localStorage.setItem('reading_game_progress', JSON.stringify({
  stageIdx: 1, lessonIdx: 0, lives: 3
}));
location.reload();
```
